# Game Assets

## Folder Structure

- `cards/monsters/` - Monster card images
- `cards/spells/` - Spell card images  
- `ui/` - UI elements (buttons, icons, etc.)

## Image Requirements

### Monster Cards
- Recommended size: 300x400px or 240x336px
- Format: PNG with transparency (preferred) or JPG
- Naming: Use card ID (e.g., `dragon.png`, `knight.png`)

### Spell Cards
- Recommended size: 300x400px or 240x336px
- Format: PNG with transparency (preferred) or JPG
- Naming: Use card ID (e.g., `lightning_bolt.png`, `heal.png`)

### Card IDs in the game:
**Monsters:**
- dragon, unicorn, knight, mage, tank, rocket, archer, golem, wyvern, berserker, paladin, necromancer, phoenix

**Spells:**
- reload, redraw, attack_boost, fort_strike, heal, star_burst, lightning_bolt, fortify, rally

## Where to Get Images

1. **Free Resources:**
   - [Unsplash](https://unsplash.com) - Free photos
   - [Pixabay](https://pixabay.com) - Free images
   - [Pexels](https://pexels.com) - Free stock photos
   - [OpenGameArt](https://opengameart.org) - Free game assets

2. **AI Image Generators:**
   - [Midjourney](https://midjourney.com)
   - [DALL-E](https://openai.com/dall-e-2)
   - [Stable Diffusion](https://stability.ai)
   - [Bing Image Creator](https://bing.com/create)
   - [Leonardo.ai](https://leonardo.ai)

3. **Icon/Art Resources:**
   - [Game Icons](https://game-icons.net) - Free game icons
   - [Flaticon](https://flaticon.com) - Free icons
   - [The Noun Project](https://thenounproject.com) - Icons

## Tips

- Use consistent art style across all cards
- Keep file sizes reasonable (< 200KB per image)
- Test images at different sizes to ensure they look good
- Consider creating card templates for consistency
